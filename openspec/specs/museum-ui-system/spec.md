# Museum UI System

## Background
The UI must evoke a high-end museum or luxury gallery feel, specifically tailored for Iranian users, utilizing a clean, minimal light-mode design.

## Requirements

### Aesthetics
- The app MUST implement the "Exaggerated Minimalism" design tokens defined by the `ui-ux-pro-max` skill.
- The app MUST use high-contrast black (`#18181B`) on white (`#FAFAFA`) as the primary color scheme.
- The app MUST use the `Inter` font for Latin characters, and a premium Persian font (e.g., Vazirmatn or Yekan Bakh) for Arabic/Persian characters.
- **CRITICAL**: All fonts and icons MUST be downloaded and stored locally in the project repository (e.g., `public/fonts/`). Do not load any CSS or font files from Google Fonts or external CDNs.

### Gamified Hook
- The hero section MUST include a scroll-tied animation where a featured item (like a high-res coin) rotates or scales dynamically as the user scrolls down, similar to Apple product pages.
